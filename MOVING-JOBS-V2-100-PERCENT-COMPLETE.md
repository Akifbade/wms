## üí∞ COST CALCULATION - FULLY AUTOMATIC

When a job is marked as **COMPLETED**, the system automatically calculates:

```
Step 1: Sum Material Costs
  = Cost/Unit √ó Quantity Used (for each material)
  Example: 0.5 KWD √ó 85 rolls = 42.5 KWD

Step 2: Sum Labor Costs
  = Hourly Rate √ó Hours Worked (for each team member)
  Example: 5 KWD/hr √ó 8 hrs = 40 KWD

Step 3: Add Other Costs
  = Fuel, equipment, misc
  Example: 50 KWD

Step 4: Calculate Total Cost
  = Material Cost + Labor Cost + Other Costs
  = 217.5 + 120 + 50 = 387.5 KWD

Step 5: Calculate Profit
  = Revenue - Total Cost
  = 800 - 387.5 = 412.5 KWD

Step 6: Calculate Margin
  = (Profit / Revenue) √ó 100%
  = (412.5 / 800) √ó 100% = 51.56%

Result: Dashboard shows all calculations immediately! ‚úì
```
