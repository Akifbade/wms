## 🚀 Use Cases

### 1. Test New Feature
```
1. Deploy code to staging
2. Test at staging.qgocargo.cloud:8080
3. Verify everything works
4. If OK, deploy to production
```

### 2. Test Database Migration
```
1. Run migration on staging database (port 3308)
2. Verify data integrity
3. If OK, run on production (port 3307)
```

### 3. Test CBM Feature
```
1. Go to staging.qgocargo.cloud:8080
2. Login
3. Create shipment
4. Test dimensions/CBM calculation
5. Verify it works
6. If OK, already in production too
```

### 4. Debug Issues
```
1. Access staging database (port 3308)
2. Run queries without affecting production
3. Test fixes
4. Deploy to production when ready
```
